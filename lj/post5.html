Game Design UX

I enjoyed reading Amir Dori's tips on mobile game UX and how applying them can benefit both users and designers. I learned the importance of considering screen size and methods to make users aware of different elements available on the page. For example, animating from end to start and scaling a popup when opening / closing. I also had not considered how timing (asking users to rate the game after a big achievement) and details such as closing a popup with a close button versus an 'x' can affect whether someone performs those actions. The main tips that I might apply to my game project are placing important buttons on the right and using animations to enhance the visibility of other important game elements.
